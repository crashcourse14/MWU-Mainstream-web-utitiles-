MWU



What is MWU?

MWU stands for "Multipurpose Web Utilities". It's written in Java and is for making HTTP Java servers easier.



Take a look

Below is a simple HTTP server that renders a HTML file from the directory /public/. It sets the port to 80 as well as it sets the public directory to "public". It automatically checks for the index.html file inside the directory you put.

It then starts the MWU server via "mwu.start()". The server will put 80 as the port by default, as well as "/public/" as the public directory if they are not declared customly.


          package src.main.java;

          import src.main.java.com.mwu.MWU;

          public class Server {
              public static void main(String[] args) {
                  MWU mwu = new MWU();

                  int port = mwu.port().setPort(80);
                  mwu.dir().setPublicDirectory("public");

                  try {
                      mwu.start();
                  } catch (Exception e) {
                      e.printStackTrace();
                  }
              }
          }
